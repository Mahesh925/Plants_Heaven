This is a plants disease detection project which detects disease based on the plant leaf image. The model is limited and trained on a select few plant species. It also displays the disease information along with prevention and treatment plan. PS. The large number of HTML percent in code is due to having a new HTML file for every plant disease info.


Install the following libraries:
pip install pandas==1.3.3 scipy==1.7.3 numpy==1.21.2 matplotlib==3.4.3 pillow==8.3.2 opencv-python==4.5.3.56 tensorflow==2.12.0 flask==2.0.1
